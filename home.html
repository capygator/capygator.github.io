CAPYGATOR BI

Local-First Business Intelligence

Open-Source BI-Plattform mit Enterprise-Patterns. Deine Daten bleiben auf deinem Rechner - keine Cloud, keine Telemetrie.

View on GitHub Quick Start
capygator@bi:~$
$ git clone https://github.com/capygator/CAPYGATOR_BI
$ cd CAPYGATOR_BI && uv sync
$ uv run dagster dev
Dagster webserver running at http://localhost:3000
$ uv run streamlit run app/main_app.py
Dashboard ready at http://localhost:8501

Features

Alles was du fuer datengetriebene Entscheidungen brauchst
Unified Dashboard
Streamlit-basiertes Dashboard mit allen deinen Daten an einem Ort. Finanz, Journal, Ernaehrung - alles vereint.
Financial Analytics
Star Schema Architektur fuer echte Analytics. Kategorien, Trends, Vertraege - alles im Blick.
Journal & Wellness
Tracke deine taeglichen Gedanken, Ernaehrung und Wohlbefinden. Mit AI-gestuetzter Analyse.
Privacy by Design
Alle Daten lokal in DuckDB. Keine Cloud-Abhaengigkeit, kein Tracking, volle Kontrolle.
