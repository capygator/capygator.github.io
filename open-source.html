Quick Start

In weniger als 5 Minuten laeuft deine eigene BI-Plattform.

Terminal
# Repository klonen
$ git clone https://github.com/capygator/CAPYGATOR_BI.git
$ cd CAPYGATOR_BI

# Dependencies installieren (mit uv)
$ uv sync

# Deine Excel-Datei platzieren
$ cp ~/Downloads/Buchungen.xlsx .

# Pipeline ausfuehren
$ uv run python sources/finanz_pipeline.py

# SQLMesh Transformationen
$ uv run sqlmesh plan --auto-apply

# Dashboard starten
$ uv run streamlit run app/main_app.py

Mit Dagster Orchestrierung

Fuer automatisierte, schedulbare Pipelines kannst du Dagster nutzen.

Terminal
# Dagster Development Server starten
$ uv run dagster dev

# Browser oeffnen: http://localhost:3000
# -> Assets -> Materialize All
